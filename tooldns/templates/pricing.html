Common Questions

What counts as a "search"?
Every time your AI agent calls POST /v1/search to find a tool. Ingesting or refreshing tools doesn't count.
Can I give clients their own keys?
Yes — use the API Key Manager to create named keys with separate usage limits. Each client gets their own key.
What's a "tool source"?
Any MCP server, skill file, or API you connect. GitHub, Slack, Gmail — each is a source with multiple tools inside.
How much does {{ app_name }} save?
Typically 95%+ of context tokens. Instead of sending 500 tool schemas (50,000+ tokens) every message, your AI gets 1-2 schemas (~200 tokens).
